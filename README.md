## What This Repository Contains

This repository provides the complete code to reproduce the results in the paper *"A Conformal Mass-Kick Sector in CLASS_EDE: First Implementation and Exploratory Constraints from Planck, BAO, and SH0ES"*.

The conformal Mass-Kick mechanism generates a time-dependent dark matter mass via a conformal coupling to a pNGB vacuum field, suppressing late-time clustering (σ₈) while preserving the EDE resolution of H₀.

## Paper Summary

Early Dark Energy (EDE) can ease the Hubble tension (H₀ = 73 vs 67 km/s/Mpc) but typically leaves the S₈ tension unaddressed. We implement a **conformal Mass-Kick coupling** — a time-dependent dark matter mass m²(χ) ∝ A²(χ) generated by a pNGB vacuum field χ — in the Boltzmann code CLASS_EDE.

**Implementation.** Three successive versions (friction-only → quasi-static screening → full Jordan-frame with CDM backreaction and fifth force) consistently show that the mechanism suppresses late-time clustering with scaling Δσ₈/σ₈ ≈ −40% × Γ × f_φ while preserving the EDE resolution of H₀.

**MCMC fit.** A 10-parameter fit against Planck 2018 TT+TE+EE+lensing, BAO (6dFGS + SDSS), and SH0ES H₀ yields:

| Parameter | Result |
|-----------|--------|
| Γ (Mass-Kick coupling) | 0.062 ± 0.043 |
| P(Γ > 0.01) | 90% |
| 95% upper limit | Γ < 0.14 |
| H₀ (mean) | 68.8 ± 1.1 km/s/Mpc |
| H₀ (best-fit) | 70.8 km/s/Mpc |
| f_EDE (mean) | 0.039 ± 0.035 |
| Best-fit −ln𝓛 | 498.3 |
| Convergence | R−1 < 0.10, 6 chains, ~880k steps |

The data show a **mild preference** for Γ > 0 but do not require it. The best-fit lies in a correlated EDE+Mass-Kick regime (H₀ ≈ 71, f_EDE ≈ 0.10, Γ ≈ 0.03) that is viable but subdominant to the ΛCDM-like solution. The coupled DM fraction f_φ is fixed at 0.20; freeing it and adding weak-lensing data (DES Y3, KiDS-1000) are the next validation steps.

**Broader context.** The Mass-Kick sector is embedded in a two-field EFT that also motivates gauge-friction inflation, U(1)_{B−L} Chern-Simons baryogenesis (η_B consistent to 𝒪(1)), and a running vacuum coefficient |α_eff| ∼ 3×10⁻³ from the dark SU(2) gauge sector. These connections are programmatic; only the Mass-Kick is tested against data in this work.

## Quick Start

### 1. Apply the Mass-Kick patch to CLASS_EDE

```bash
# Clone CLASS_EDE
git clone https://github.com/mwt5345/class_ede.git
cd class_ede

# Apply the Mass-Kick patch
git apply ../patch/v8_mass_kick_v2.patch

# Build (Python 3.12 requires the Cython fix)
bash ../build/fix_classy_v5.sh
```

### 2. Run a test spectrum

```bash
cd ..
cobaya-run mcmc/v8_mcmc_diag.yaml --test
```

### 3. Reproduce the MCMC fit

```bash
# 6-chain production run (requires ~6 CPU cores, ~8 GB RAM, ~1 week)
mpirun -n 6 cobaya-run mcmc/v8_mcmc_final.yaml
```

### 4. Generate paper figures

```bash
python3 analysis/make_paper_figures.py
```

## The Mass-Kick Mechanism

The conformal coupling modifies the CDM energy density:

```
A²(χ) = 1 + (Γ/2) cos(χ/f_χ)
```

This enters CLASS_EDE through:
- **background.c**: Modified CDM background evolution with conformal factor
- **perturbations.c**: Modified CDM perturbation equations (density + velocity + fifth force)

### Parameters

| Parameter | Description | Prior |
|-----------|-------------|-------|
| `Gamma_mk` | Mass-Kick coupling strength | [0, 0.50] |
| `f_phi_mk` | Coupled DM fraction | Fixed at 0.20 |

### Null Test

Setting `Gamma_mk = 0` reproduces standard EDE spectra to < 0.01% in C_ℓ.

## Build Notes (Python 3.12 + Cython 3.x)

CLASS_EDE was designed for Python 3.8-3.10. On Python 3.12, the Cython build fails. The fix script `fix_classy_v5.sh` handles:

1. Downgrade Cython to < 3.0
2. Fix `np.int_t` → `np.int64_t`
3. Add `# cython: language_level=3`
4. Fix Python 2 dict methods (`.iteritems()` etc.)
5. Insert `#undef I` between NumPy and CLASS headers
6. Direct gcc compilation of the `.pyx` → `.c` → `.so`
